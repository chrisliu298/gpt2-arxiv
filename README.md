# CSE142 Project6: GPT-2-arXiv

This GPT-2 model is capable of generating abstracts given paper titles.

We will add argparse support in later versions.


## Getting Started

### Prerequisites

```
> pip install -r requirements.txt
```

## Fine-Tune

```
> python train.py
```

## Generate

```
> python generate.py
```
